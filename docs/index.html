🩺 Diabetic Retinopathy Detector

Upload a retinal fundus photo. It predicts a severity grade (0–4), shows where the model looked, and highlights the classic lesions — all in your browser.

⚠️ Not a medical device. Expects proper fundus (back-of-the-eye) photographs, not ordinary photos of an eye. Everything runs locally in your browser — no image is uploaded anywhere.

About

Two models run entirely on your device: an EfficientNet classifier for the 0–4 grade and a U-Net that segments four lesion types. Source code and training pipeline: GitHub repository. If analysis reports missing models, the .onnx files still need to be added to docs/models/ (see the repo's deployment guide).
